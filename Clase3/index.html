Listas

Las listas son un conjunto de elementos que se muestran en forma de lista.

Lista ordenada

La lista ordenada es aquella en la que los elementos se muestran en forma de lista numerada.

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

Lista desordenada

La lista desordenada es aquella en la que los elementos se muestran en forma de lista con viñetas.

  • Elemento 1
  • Elemento 2
  • Elemento 3

Lista de definición

La lista de definición es aquella en la que los elementos se muestran en forma de lista con un término y una descripción.

Termino 1
Descripción del elemento 1
Termino 2
Descripción del elemento 2
Termino 3
Descripción del elemento 3

Lista anidada

La lista anidada es aquella en la que los elementos se muestran en forma de lista con elementos anidados.

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4
    • Elemento 4.1
    • Elemento 4.2
    • Elemento 4.3
